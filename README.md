## 📖 Sobre o Projeto

O **AuraUP** nasceu para dar visibilidade a profissionais autônomos e microempresas. Combinamos a estética visual do **Instagram** (foco no portfólio e imagem) com a praticidade da **OLX** (negociação e encomenda de serviços). 

Nossa missão é facilitar o encontro entre quem sabe fazer e quem precisa contratar, tudo em um ambiente intuitivo e moderno.

### 🚀 Funcionalidades Principais
- [x] **Vitrine Digital:** Exposição de serviços com fotos e descrições.
- [x] **Sistema de Encomendas:** Solicite orçamentos e serviços diretamente pela plataforma.
- [x] **Perfis Personalizados:** Cada prestador tem seu "espaço Aura" para mostrar sua marca.
- [x] **Busca Inteligente:** Filtre serviços por categoria e localização.
